It is necessary, if linaro toolchain for ARM is installed, to use gdb-multiarch as a debbuger in 
QT-Creator. QT-Creator will NOT recognize the binary delivered with the linaro toolchain.

To be able to start QT Creator on Ubuntu 20.4, an additional library needs to be installed:

sudo apt-get install libxcb-xinerama0

Otherwhise QT-Creator will not start (Even when setup with the QT-Installer)
